# Private AI Assistant with Tool Use

A locally-run AI assistant (via Ollama) that can chat, search your documents (RAG), 
and fill PDF forms — with no data leaving your machine.

## Status: 🚧 In progress

## Planned features
- [ ] Chat with local LLM via Ollama
- [ ] Tool-calling agent loop
- [ ] PDF reading + form filling
- [ ] Document RAG search
- [ ] Web UI
